## 当前核心研究问题

> 在 RAG/Agent 请求类型差异较大的场景下，如何通过任务调度降低尾延迟并提升 worker 利用率？

当前先把这个问题收束到 [[60_科研训练/研究项目/RQ01_RAG_Agent请求调度尾延迟/00_研究问题|RQ01]]，后续论文、实验设计、实验记录和写作草稿都围绕这个研究项目沉淀。

## 就业表达出口

科研训练中的内容进入简历、面试或导师沟通前，必须先经过 [[80_就业市场与简历/简历项目表达/项目与科研表达模板_按完成状态分级|项目与科研表达模板：按完成状态分级]] 检查。

RQ01 已有可重放的 30-seed E2 synthetic pilot 与报告，只能表达限定合成条件下的观察。
Pareto/burst、clean final、P03 服务回放和真实 RAG/Agent trace 尚未完成，不得写成场景、生产或普遍结论。

## 训练流程

```text
工程现象
-> 研究问题
-> baseline
-> 实验变量
-> 指标
-> 结果图表
-> 结论
-> 局限
-> 下一步
```
