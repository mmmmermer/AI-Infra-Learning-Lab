## 调度与尾延迟基础

| 资料 | 需要提取的问题 | 与 RQ01 的关系 |
|---|---|---|
| [Schrage, A Proof of the Optimality of the Shortest Remaining Processing Time Discipline](https://doi.org/10.1287/opre.16.3.687) | SRPT 的假设、抢占条件和目标函数 | 防止把非抢占 predicted SJF 与 SRPT 最优性混同 |
| [Dean and Barroso, The Tail at Scale](https://doi.org/10.1145/2408776.2408794) | 为什么尾延迟影响大规模服务 | 支撑研究 P95/P99，但不直接证明调度策略有效 |

| [Kueue Documentation](https://kueue.sigs.k8s.io/docs/) | workload admission、quota、LocalQueue/ClusterQueue | 区分 batch admission 与进程内任务排序 |

## LLM serving 与请求调度

| 资料 | 需要提取的问题 | 与 RQ01 的关系 |
|---|---|---|
| [Orca: A Distributed Serving System for Transformer-Based Generative Models](https://www.usenix.org/conference/osdi22/presentation/yu) | iteration-level scheduling、请求混合 | 说明生成请求长度差异与 serving 调度相关 |
| [vLLM: Easy, Fast, and Cheap LLM Serving with PagedAttention](https://arxiv.org/abs/2309.06180) | KV cache、continuous batching、吞吐 | RQ01 后续不能忽略 serving engine 内部 batching |
| [FastServe: Fast Distributed Inference Serving for Large Language Models](https://www.usenix.org/conference/atc23/presentation/wu) | preemption、skip-join MLFQ、JCT | 与非抢占 predicted SJF 比较边界 |
| [Sarathi-Serve](https://arxiv.org/abs/2403.02310) | chunked prefill、吞吐-延迟权衡 | 提醒 prompt/decode 阶段资源特征不同 |
| [DistServe](https://arxiv.org/abs/2401.09670) | prefill/decode 分离、goodput | 说明真实 serving 不是单一 service duration |
| [Llumnix](https://arxiv.org/abs/2406.03243) | 动态调度、迁移和负载变化 | 为 burst、迁移和多实例扩展提供 prior art |
| [vLLM Metrics](https://docs.vllm.ai/en/latest/usage/metrics.html) | 可观测指标和版本差异 | P03 回放阶段的服务端指标出口 |
