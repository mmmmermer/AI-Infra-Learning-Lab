# RQ01 实验报告

> 状态：`E2 pilot report / final phase B and E3-E4 blocked`。当前可以报告指定合成条件下的配对观察，但不能写成 RAG/Agent、生产系统或普遍策略结论。

## 摘要

30-seed 合成 pilot 使用开放环 Poisson 到达、lognormal short/medium/long mixture、
单 worker 和三档乘性预测误差，对比 FIFO、predicted SJF 与 oracle SJF。predicted SJF
相对 FIFO 降低 P95 wait，但提高 P99 wait 和 long-class P99 wait；预测误差增大时这种
尾部伤害扩大。该结果仅是 E2 pilot tradeoff 证据，尚未覆盖 burst/Pareto、最终 seed
规则、P03 服务回放或真实 RAG/Agent trace。

## 当前可写观察

在本 pilot 的指定单 worker、高负载、Poisson/lognormal 合成条件下，size-based
non-preemptive scheduling 呈现明显 percentile/fairness tradeoff：P95 改善不意味着
P99 改善，长任务尾部可能显著受损；预测误差升高会削弱 predicted SJF 的 P95 收益并
扩大 P99 伤害。下一步必须用 Pareto/burst 条件和 P03 固定到达回放检验该观察是否稳健。
