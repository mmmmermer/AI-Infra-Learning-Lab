# P01 v0.1 阶段执行说明

## 本阶段目标

用 3-5 天做出一个纯 Python 版 Mini Scheduler。完成后，你应该能运行一个 demo，看到 FIFO、Priority、SJF 三种策略对任务顺序和等待时间的影响。

这不是最终项目，而是后续 FastAPI、RAG、监控、压测、科研实验的最小内核。

## 遇到卡点怎么处理

### 不知道代码放哪里

先看 [[50_项目产出/P01_Mini_Scheduler/03_代码结构与接口]]。原则是：模型放 `models.py`，策略放 `strategies.py`，模拟执行放 `simulator.py`，指标放 `metrics.py`。

### pytest 找不到模块

优先检查：

- 当前工作目录是不是项目根目录
- `scheduler/__init__.py` 是否存在
- 测试文件是否在 `tests/` 下
- import 路径是否写成 `from scheduler...`

### 指标算出来很奇怪

先手算一个只有 2-3 个任务的小例子。调度系统里很多 bug 都不是语法错，而是时间线想错了。
