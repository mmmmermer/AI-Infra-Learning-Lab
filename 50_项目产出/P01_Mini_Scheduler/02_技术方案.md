# P01 技术方案

## 核心对象

### Task

- id
- task_type
- priority
- estimated_duration
- submit_time
- start_time
- finish_time
- status

### Worker

- id
- available_at
- current_task
- total_busy_time

### Scheduler

- strategy
- queue
- assign_task
- collect_metrics

## 第一版策略

- FIFO：按提交顺序执行
- Priority：按优先级执行
- SJF：按预计耗时从短到长执行

## 指标

- 平均等待时间
- P95 等待时间
- 平均完成时间
- 队列长度
- worker 利用率
