# E00 工具链基础实验索引

## 定位

E00 是 M00 的实操区，用来训练自学工程项目最基本的工具能力。

这一组实验不追求复杂功能，而是让你形成一条最小闭环：创建项目、进入目录、运行代码、安装依赖、记录日志、写测试、用 Git 保存进度。

## 推荐实验

- [[40_实验练习/E00_工具链基础实验/E00-01 命令行创建 Python 项目|E00-01 命令行创建 Python 项目]]：训练目录、路径、终端和 Python 运行
- [[40_实验练习/E00_工具链基础实验/E00-02 Git 保存学习进度|E00-02 Git 保存学习进度]]：训练 Git 保存和查看学习进度
- [[40_实验练习/E00_工具链基础实验/E00-03 用 httpx 请求一个 API|E00-03 用 httpx 请求一个 API]]：训练 HTTP、JSON 和网络请求
- [[40_实验练习/E00_工具链基础实验/E00-04 写一个带日志的小脚本|E00-04 写一个带日志的小脚本]]：训练 logging 和问题定位
- [[40_实验练习/E00_工具链基础实验/E00-05 为任务排序函数写 pytest|E00-05 为任务排序函数写 pytest]]：训练最小测试和规则验证

## 建议顺序

1. 先做 E00-01，确保能独立创建并运行 Python 文件
2. 再做 E00-02，把第一次实验用 Git 保存下来
3. 再做 E00-04，把日志加入到 `task_sorter`
4. 再做 E00-05，用 pytest 固定排序规则
5. 最后做 E00-03，理解后续 FastAPI/RAG 会用到的 HTTP/JSON
