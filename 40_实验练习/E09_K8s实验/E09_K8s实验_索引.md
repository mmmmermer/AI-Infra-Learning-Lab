> 状态（2026-07-11，当前 P03 契约 v0.3.1）：E09-01 至 E09-03 已有 kind v0.32.0 / Kubernetes v1.34.8 可执行本机 reference：完成五类 Deployment、RAG 权限、API rollout 持久查询和 1/2/4 worker 功能性扩容。E09-04 仍是 Kueue 概念实验；metrics-server/HPA、持久存储、重复集群实验和学习者复现仍未完成。

## 定位

E09 用于长期训练 Kubernetes 基础部署、worker Pod 扩缩容和 AI workload 调度概念映射。

第一轮不是生产部署实验，而是把 M09 导论教材落到最小可观察实验：

```text
Docker Compose 项目
-> 本地 K8s 集群
-> API Deployment / Service
-> worker replicas
-> M05 调度概念映射
```

## 最小练习路线

```text
先能启动本地集群
-> 部署一个 FastAPI API
-> 给 API 配 Service
-> 用 ConfigMap/Secret 注入配置
-> 模拟 worker replicas 改变
-> 对照 M05 解释 Task/Worker/Queue/Scheduler
```
