## 实验步骤

### 步骤 1：建立收益率样本

```python
import pandas as pd

df = pd.DataFrame(
    {
        "date": [
            "2024-01-02", "2024-01-03", "2024-01-04", "2024-01-05",
            "2024-01-08", "2024-01-09", "2024-01-10", "2024-01-11",
            "2024-01-12", "2024-01-15",
        ],
        "asset": ["TEST"] * 10,
        "return": [0.010, -0.008, 0.006, 0.012, -0.025, 0.004, 0.009, -0.006, 0.015, -0.018],
    }
)

df["date"] = pd.to_datetime(df["date"])
returns = df["return"]
```

### 步骤 2：计算分布摘要

```python
summary = {
    "sample_size": len(returns),
    "mean": returns.mean(),
    "std": returns.std(),
    "min": returns.min(),
    "max": returns.max(),
    "q05": returns.quantile(0.05),
    "q50": returns.quantile(0.50),
    "q95": returns.quantile(0.95),
}

print(summary)
```

### 步骤 3：观察直方图

```python
ax = returns.hist(bins=5)
ax.set_title("Synthetic return distribution")
```

如果当前环境不方便画图，也可以只记录分位数和极端值。图表不是目的，解释才是目的。

### 步骤 4：写 normality_note

示例：

```text
样本只有 10 个自造收益率，不能判断真实分布。直方图只用于观察流程。即使形状接近钟形，也不能证明收益率服从正态分布。
```

## 观察指标

| 指标 | 含义 | 记录要求 |
|---|---|---|
| sample_size | 样本数量 | 必填 |
| mean | 样本均值 | 必填 |
| std | 样本标准差 | 必填 |
| min / max | 样本极端值 | 必填 |
| q05 / q50 / q95 | 分位数 | 必填 |
| histogram_bins | 直方图分箱数 | 如画图则记录 |

## 常见错误

- 用 10 个样本断言真实收益率分布。
- 只看 mean，不看 std 和极端值。
- 把自造数据写成真实资产结论。
- 看到直方图像钟形，就说“服从正态分布”。
- 不记录 sample_size 和 frequency。
