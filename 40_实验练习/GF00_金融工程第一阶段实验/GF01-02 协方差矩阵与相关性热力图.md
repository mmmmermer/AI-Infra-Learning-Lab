
## 实验步骤

### 步骤 1：建立收益率矩阵

```python
import pandas as pd

returns_df = pd.DataFrame(
    {
        "date": [
            "2024-01-02", "2024-01-03", "2024-01-04",
            "2024-01-05", "2024-01-08", "2024-01-09",
        ],
        "A": [0.010, -0.020, 0.015, 0.005, -0.010, 0.012],
        "B": [0.008, -0.015, 0.012, 0.004, -0.006, 0.009],
        "C": [-0.002, 0.004, -0.001, 0.002, 0.003, -0.004],
    }
)

returns_df["date"] = pd.to_datetime(returns_df["date"])
returns_df = returns_df.set_index("date")

asset_order = ["A", "B", "C"]
returns_df = returns_df[asset_order]
```

### 步骤 2：计算协方差和相关性

```python
cov_matrix = returns_df.cov()
corr_matrix = returns_df.corr()

print(cov_matrix)
print(corr_matrix)
```

### 步骤 3：画相关性热力图

如果环境安装了 seaborn：

```python
import seaborn as sns
import matplotlib.pyplot as plt

sns.heatmap(corr_matrix, annot=True, vmin=-1, vmax=1, cmap="coolwarm")
plt.title("Synthetic correlation matrix")
plt.show()
```

如果没有 seaborn，只记录 `corr_matrix` 表格即可。

### 步骤 4：写解释说明

合格解释示例：

```text
在这组自造样本中，A 和 B 的相关性较高，说明它们在样本内经常同向波动；C 与 A/B 的相关性较低或相反，可能降低组合波动。但样本是自造数据，不能代表真实资产关系，也不能推断未来相关性稳定。
```

## 常见错误

- 不记录 `asset_order`。
- 把相关性写成因果关系。
- 把样本内低相关写成未来稳定分散风险。
- 忽略缺失值处理。
- 用相关矩阵直接代替协方差矩阵计算组合方差。
