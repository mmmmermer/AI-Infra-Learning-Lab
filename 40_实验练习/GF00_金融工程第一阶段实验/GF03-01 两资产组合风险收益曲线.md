## 为什么这个实验有意义

单资产风险只看一个收益率序列。组合风险要同时看：

- 每个资产自己的波动。
- 两个资产的权重。
- 两个资产之间是否一起涨跌。

如果两个资产相关性较低，组合波动可能低于单纯加权平均波动；如果两个资产高度同向，分散效果就弱。

这个实验的重点不是找到“最优权重”，而是理解：

```text
组合风险来自权重、单资产波动和资产之间的共同波动。
```

## 知识点与难点

### 组合收益

两资产组合收益：

```text
r_p = w_A * r_A + w_B * r_B
```

第一轮只做 long-only 组合：

```text
w_A >= 0
w_B >= 0
w_A + w_B = 1
```

### 组合波动

组合波动不是：

```text
w_A * vol_A + w_B * vol_B
```

而是由协方差矩阵决定：

```text
portfolio_variance = w^T Sigma w
portfolio_volatility = sqrt(portfolio_variance)
```

### 工程痛点

组合实验最容易出错的地方：

- A/B 两列日期没有对齐。
- 权重顺序和收益率列顺序不一致。
- 权重和不等于 1。
- 忘记记录是否允许卖空。
- 把曲线上某一点写成“最佳组合”。
- 不说明样本太小、数据是自造的。

### 步骤 3：枚举权重

本实验枚举 A 的权重从 0 到 1，B 的权重等于 `1 - w_A`。

```python
weight_grid = np.linspace(0, 1, 11)
```

记录：

```text

```

### 步骤 4：计算组合收益和组合波动

```python
rows = []
asset_order = ["A", "B"]
mu = returns_df[asset_order].mean().values
sigma = returns_df[asset_order].cov().values

for w_a in weight_grid:
    weights = np.array([w_a, 1 - w_a])
    portfolio_return = float(weights @ mu)
    portfolio_variance = float(weights @ sigma @ weights)
    portfolio_volatility = portfolio_variance ** 0.5
    rows.append(
        {
            "weight_A": float(w_a),
            "weight_B": float(1 - w_a),
            "portfolio_return": portfolio_return,
            "portfolio_volatility": portfolio_volatility,
        }
    )

portfolio_table = pd.DataFrame(rows)
```

## 观察指标

| 指标 | 含义 | 解释边界 |
|---|---|---|
| `portfolio_return` | 给定权重下的样本平均组合收益 | 不代表未来收益 |
| `portfolio_volatility` | 给定权重下的样本组合波动 | 受协方差和样本影响 |
| `corr(A,B)` | 两资产收益共同波动关系 | 小样本下不稳定 |
| `weight_A/weight_B` | 组合权重 | 本实验 long-only，权重和为 1 |

## 常见错误

- 权重和不等于 1。
- 权重顺序和收益率列顺序不一致。
- 只看组合收益，不看组合波动。
- 忽略相关性和协方差。
- 把某个点解释成“最优投资组合”。
- 用自造数据写真实市场结论。
- 不记录是否允许卖空和杠杆。
