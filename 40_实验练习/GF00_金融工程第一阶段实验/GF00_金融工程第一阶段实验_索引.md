
## 第一阶段实验总路线

第一阶段实验只服务两个目标：

1. 能处理并解释金融时间序列数据。
2. 能计算基础风险指标，并把任务抽象成未来 P03 workload。

路线如下：

```text
数据来源和字段口径
-> 时间序列清洗
-> 收益率和波动率计算
-> 协方差和相关性
-> 回撤、VaR、组合风险
-> risk_metric_task / portfolio_task 草图
```

## 第一轮优先级

第一轮具体执行表见：

- [[00_路线总控/看板与索引/10_F系列第一阶段实验执行表|F 系列第一阶段实验执行表]]

第一轮不建议一次性执行所有实验。

如果后续正式进入金融工程学习，优先顺序是：

1. GF02-01 金融时间序列清洗。
2. GF02-02 rolling volatility 与最大回撤。
3. GF03-03 VaR 与最大回撤计算。
4. GF03-01 两资产组合风险收益曲线。

原因：

- 它们最容易从 F02/F03 直接练起来。
- 它们能支撑 Q01/Q02 的最小项目输出。
- 它们可以直接映射到 P03 的 `risk_metric_task`。

可选前置：

- GF01-01 收益率分布与正态性观察：如果 F03 的 VaR 学起来吃力，先做它。
- GF01-02 协方差矩阵与相关性热力图：如果 F03 的组合风险学起来吃力，先做它。

暂缓：

- 任何期权、固定收益、回测、机器学习实验。

## 最小数据要求

第一阶段不要求购买商业数据，也不要求使用真实交易系统。

允许使用：

- 公开数据源的小样本数据。
- 自造测试数据。
- 从官方 API 文档中获取的演示数据。
- 明确标注来源和限制的免费数据。

每个实验必须记录：

```text
data_source
source_url
asset_type
symbol_or_series_id
date_range
frequency
fields
adjusted_or_raw
missing_ratio
license_or_access_note
```
