# E05 调度实验索引

## 定位

用于训练任务队列、worker 分配、调度策略、等待时间、P95/P99 和资源利用率。

E05 是 M05 章节教材的配套实验，不是独立项目台账。学习顺序应当是：

```text
先读章节教材 -> 亲手写最小代码 -> 再做对应实验 -> 最后对照 P01 参考答案
```
