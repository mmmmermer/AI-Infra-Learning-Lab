本实验承接 GF06-01，目标是观察交易成本如何改变回测结果。

它不是为了模拟真实交易系统，而是训练一个关键习惯：

```text

```

## 实验步骤

### 步骤 1：计算仓位变化

```python
df["position_change"] = (
    df["position_at_open"]
    .diff()
    .abs()
    .fillna(df["position_at_open"].abs())
)
df["turnover"] = df["position_change"]
```

### 步骤 2：加入固定成本

```python
cost_rate = 0.001

df["transaction_cost"] = df["position_change"] * cost_rate
df["net_strategy_return"] = df["strategy_return"] - df["transaction_cost"]
```

### 步骤 3：比较无成本和有成本净值

```python
df["gross_equity_curve"] = (1 + df["strategy_return"].fillna(0)).cumprod()
df["net_equity_curve"] = (1 + df["net_strategy_return"].fillna(0)).cumprod()

gross_final = df["gross_equity_curve"].iloc[-1]
net_final = df["net_equity_curve"].iloc[-1]
avg_turnover = df["turnover"].mean()
```

### 步骤 4：做成本敏感性观察

```python
for cost_rate in [0.0, 0.001, 0.003]:
    cost = df["position_change"] * cost_rate
    net_ret = df["strategy_return"] - cost
    final_equity = (1 + net_ret.fillna(0)).cumprod().iloc[-1]
    print(cost_rate, final_equity)
```

## 常见错误

- 只展示无成本结果。
- 不记录 cost_rate。
- 把固定成本模型说成真实交易成本。
- 不计算 turnover。
- 成本扣减方向写反。
