## 实验定位

本实验用利率上升/下降的数值实验理解债券价格敏感性。它可以作为 P03 `fixed_income_task` 的敏感性分析前置练习，但不产生债券投资建议。

第一轮不推导复杂公式，只观察：

```text
yield up -> price down
yield down -> price up
```

并用 duration/convexity 直觉描述这种关系。

### 步骤 2：设置利率冲击

```python
base_yield = 0.04
shock = 0.01

payments_per_year = 2
price_base = present_value(cashflows, base_yield, payments_per_year)
price_up = present_value(cashflows, base_yield + shock, payments_per_year)
price_down = present_value(cashflows, base_yield - shock, payments_per_year)

print(price_base, price_up, price_down)
```

### 步骤 3：计算久期与凸性

```python
def macaulay_duration(cashflows, annual_yield, payments_per_year=2):
    price = present_value(cashflows, annual_yield, payments_per_year)
    periodic_yield = annual_yield / payments_per_year
    weighted_pv = sum(
        time_years * cf / ((1 + periodic_yield) ** period)
        for period, time_years, cf in cashflows
    )
    return weighted_pv / price


def modified_duration(cashflows, annual_yield, payments_per_year=2):
    return macaulay_duration(cashflows, annual_yield, payments_per_year) / (
        1 + annual_yield / payments_per_year
    )


def convexity(cashflows, annual_yield, payments_per_year=2):
    price = present_value(cashflows, annual_yield, payments_per_year)
    periodic_yield = annual_yield / payments_per_year
    numerator = sum(
        cf * period * (period + 1) / ((1 + periodic_yield) ** (period + 2))
        for period, _time_years, cf in cashflows
    )
    return numerator / (price * payments_per_year**2)


macaulay = macaulay_duration(cashflows, base_yield, payments_per_year)
modified = modified_duration(cashflows, base_yield, payments_per_year)
analytic_convexity = convexity(cashflows, base_yield, payments_per_year)

effective_duration = (price_down - price_up) / (2 * price_base * shock)
effective_convexity = (price_down + price_up - 2 * price_base) / (price_base * shock**2)

estimated_up_change = -modified * shock + 0.5 * analytic_convexity * shock**2
actual_up_change = (price_up - price_base) / price_base
```

## 常见错误

- 以为 duration 是准确预测。
- 忽略利率变化幅度。
- 不记录 base_yield。
- 只做上下冲击，没有实际计算久期或凸性。
- 混淆 Macaulay duration 与 modified duration。
- 忘记把付息频率纳入贴现和凸性年化口径。
- 把利率敏感性实验写成债券买卖建议。
