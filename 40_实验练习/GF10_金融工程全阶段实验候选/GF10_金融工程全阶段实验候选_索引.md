# GF10 金融工程全阶段实验候选索引

## 定位

GF10 是金融工程第二主线的全阶段实验索引。

它不是已完成项目成果。它的作用是把 F00-F08 的学习内容转成后续可以亲手练习、记录、复盘、接入 P03 的实验方向；其中 GF04-GF08 已经补出主要实验详情页，仍然只代表“实验说明已建好”，不代表实验已经亲手完成。

```text
F00-F08 学习地图
-> GF10 实验候选
-> Q01-Q04 项目候选
-> F08 / X02 / P03 workload
```

## 第一轮优先级

因为你基础相对薄弱，第一轮不要从期权或机器学习开始。

1. GF02-01 金融时间序列清洗。
2. GF02-02 rolling volatility 与最大回撤。
3. GF03-03 VaR 与最大回撤计算。
4. GF03-01 两资产组合风险收益曲线。
5. [[40_实验练习/GF10_金融工程全阶段实验候选/GF06-01 均线策略最小回测|GF06-01 均线策略最小回测]]。

## 验收标准

- [ ] 每个候选实验都能找到来源模块。

- [ ] 每个候选实验都能写出指标和记录字段。
- [ ] 至少一部分实验能映射到 P03 task。
- [ ] 不把候选实验写成已完成项目。
- [ ] 不把回测或模型结果写成投资建议。
