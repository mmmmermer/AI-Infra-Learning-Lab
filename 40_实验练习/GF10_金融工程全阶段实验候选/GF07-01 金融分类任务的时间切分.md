## 实验定位

本实验服务 F07 的第一步：把金融时间序列构造成一个最小分类任务，并按时间顺序切分训练集和测试集。

它不是为了训练强模型，而是为了理解：

```text
features_t 只能使用 t 时刻可见信息
label_t 可以来自 t+1
train 必须早于 test
```

## 实验步骤

### 步骤 1：建立价格和收益率

```python
import pandas as pd

df = pd.DataFrame(
    {"date": pd.bdate_range("2024-01-02", periods=60)}
)

return_pattern = [0.01, -0.008, 0.012, -0.006, 0.004, -0.011, 0.009, -0.003]
prices = [100.0]
for index in range(1, len(df)):
    period_return = return_pattern[(index - 1) % len(return_pattern)]
    prices.append(prices[-1] * (1 + period_return))
df["adjusted_close"] = prices

df["date"] = pd.to_datetime(df["date"])
df = df.sort_values("date").set_index("date")
df["return"] = df["adjusted_close"].pct_change()
```

### 步骤 2：构造 label

```python
df["future_return"] = df["return"].shift(-1)
df = df.dropna(subset=["future_return"]).copy()
df["label"] = (df["future_return"] > 0).astype("int8")
```

`future_return` 只能用于生成 label，不能进入 feature。若先比较再转整数，末行 NaN 会被错误变成 0，后续 `dropna(label)` 无法清除这个假标签。

### 步骤 3：构造历史特征

```python
df["return_lag_1"] = df["return"].shift(1)
df["return_lag_2"] = df["return"].shift(2)
df["rolling_vol_3"] = df["return"].rolling(3).std().shift(1)

features = ["return_lag_1", "return_lag_2", "rolling_vol_3"]
dataset = df.dropna(subset=features).copy()
```

### 步骤 4：按时间顺序切分并留 gap

```python
horizon = 1
train_end = int(len(dataset) * 0.6)
validation_end = int(len(dataset) * 0.8)

train = dataset.iloc[:train_end]
validation = dataset.iloc[train_end + horizon : validation_end]
test = dataset.iloc[validation_end + horizon :]

assert not train.empty and not validation.empty and not test.empty
assert train["label"].nunique() == 2
assert train.index.max() < validation.index.min() < test.index.min()

print(train.index.min(), train.index.max())
print(validation.index.min(), validation.index.max())
print(test.index.min(), test.index.max())
```

`validation` 用于选择特征、阈值和模型参数；`test` 只在方案冻结后评估一次。样本太小导致任一集合为空时，实验应明确失败并扩大数据，而不是取消 gap。

## 常见错误

- 把 `future_return` 放进 features。
- 随机打乱时间序列。
- rolling 特征没有 shift。
- 不记录 label 的预测 horizon。
- 使用 test 选择特征、阈值或超参数。
- 样本太小时取消 gap 或重复查看 test 结果。
- 用自造数据写真实市场结论。
