# GF06-01 均线策略最小回测

## 实验定位

本实验服务 F06，不是为了寻找可交易策略，而是练习最小回测流程：

```text
adjusted open / close
-> open-to-close return
-> moving_average
-> signal_at_close
-> position_at_open = signal_at_close.shift(1)
-> strategy_return
-> equity_curve
-> backtest_task 草图
```

重点是理解 signal、position 和 return 的时间关系，避免未来信息泄漏。

## 实验步骤

### 步骤 1：建立价格序列

```python
import pandas as pd

df = pd.DataFrame(
    {
        "date": [
            "2024-01-02", "2024-01-03", "2024-01-04", "2024-01-05",
            "2024-01-08", "2024-01-09", "2024-01-10", "2024-01-11",
            "2024-01-12", "2024-01-15",
        ],
        "adjusted_open": [
            100.0, 100.5, 100.0, 100.5, 102.0,
            102.0, 102.0, 104.0, 104.0, 104.0,
        ],
        "adjusted_close": [100, 101, 99, 102, 103, 101, 104, 105, 103, 106],
    }
)

df["date"] = pd.to_datetime(df["date"])
df = df.sort_values("date").set_index("date")
```

### 步骤 2：计算收益率和均线

```python
window = 3

df["open_to_close_return"] = df["adjusted_close"] / df["adjusted_open"] - 1
df["ma"] = df["adjusted_close"].rolling(window=window).mean()
```

### 步骤 3：收盘生成 signal，次日开盘形成 position

```python
df["signal_at_close"] = (df["adjusted_close"] > df["ma"]).astype(int)
df["position_at_open"] = df["signal_at_close"].shift(1).fillna(0)
```

这里必须解释：

```text
decision_time = close t
fill_time = open t+1
return_interval = open t+1 to close t+1
```

`shift(1)` 只完成表格对齐；上面三个时点字段共同定义可成交的教学时钟。

### 步骤 4：计算策略收益和净值曲线

```python
df["strategy_return"] = df["position_at_open"] * df["open_to_close_return"]
df["equity_curve"] = (1 + df["strategy_return"].fillna(0)).cumprod()

print(
    df[
        [
            "adjusted_open", "adjusted_close", "ma", "signal_at_close",
            "position_at_open", "open_to_close_return", "strategy_return",
            "equity_curve",
        ]
    ]
)
```

## 常见错误

- 用收盘信号直接乘同日开盘到收盘收益。
- 不说明 signal 什么时候生成、position 什么时候生效。
- 不记录 rolling window。
- 把自造数据的结果写成真实策略效果。
- 只保存净值曲线，不保存参数和限制。
