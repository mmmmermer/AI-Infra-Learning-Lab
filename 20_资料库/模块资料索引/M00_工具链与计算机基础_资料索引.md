# M00 工具链与计算机基础资料索引

## 当前策略

M00 的资料只服务一个目标：让你能独立创建、运行、调试、测试、保存一个最小 Python 小项目。

这不是计算机基础资料收藏夹。当前阶段不追求完整学完 Shell、Git、HTTP、操作系统或网络课程，只选能直接支撑 M00 教材和 E00 实验的部分。

## 当前只读哪些部分

### MIT Missing Semester

当前只看：

- Shell 是什么
- 命令如何在当前目录执行
- 文件、目录、路径的基本概念
- Git 的基本模型

暂时不看深：

- 高级 shell 脚本
- vim 深度配置
- shell pipeline 高级组合
- dotfiles 管理

### Python 官方文档

当前只看：

- `venv` 如何创建和激活
- `python -m pip` 的使用方式
- `logging.basicConfig`
- `logging.info/error/exception`

暂时不看深：

- logging handler/filter/formatter 的完整体系
- Python 打包发布
- 多版本环境管理生态比较

### pytest 官方文档

当前只看：

- 测试文件命名
- `assert`
- 如何运行 `pytest`
- 失败信息怎么看

暂时不看深：

- fixture 高级用法
- parametrize 大量组合
- mock
- plugin 生态

### HTTP / HTTPX

当前只看：

- GET 请求
- status code
- JSON response
- 超时和异常的直觉

暂时不看深：

- 认证授权
- HTTP/2
- cookie/session
- 连接池和高级客户端配置

M00 每条必读资料至少要转化成一种产出：

- 一条知识卡片
- 一页实验记录
- 一个项目功能
- 一段 README 说明

当前优先转化：

- Shell / 路径 -> E00-01
- Git commit -> E00-02
- HTTP / JSON -> E00-03
- logging -> E00-04
- pytest -> E00-05
