## 这个库的定位

它不是纯学习资料库，也不是纯路线规划库。

如果你打开学习模块后感觉像台账，先看 [[10_学习模块/00_学习模块阅读指南|学习模块阅读指南]]，按“为什么学 -> 解决什么项目问题 -> 最小例子 -> 练习 -> 检查标准”的方式读，不要把所有资料索引都当成待办清单。

它是一套从路线到产出的系统：

```text
路线规划
-> 学习模块
-> 资料库
-> 知识卡片
-> 实验练习
-> 项目产出
-> 科研训练
-> 导师联系
-> 就业简历
-> 计划复盘
```

## 后续填充原则

以后新增资料时，不要只放链接。至少要归入：

- 一个学习模块
- 一个资料卡
- 一个可提炼的知识卡片
- 一个可做的实验
- 一个项目应用位置
