# F05 固定收益与利率基础学习地图

## 怎么读这个模块

把 F05 当成金融工程第二主线里的“现金流、折现和利率风险”模块。

第一轮只需要理解：

```text
债券现金流
-> 折现和到期收益率
-> 久期和凸性
-> 利率变化如何影响价格
```

它不是复杂利率模型课程，也不是债券交易课程。它的作用是帮你理解固定收益数据、利率曲线和风险指标，后续可以连接 F03 风险管理、F04 定价参数、F08 任务接口和 P03 的长期计算任务。

## 在总路线中的位置

```text
F00 金融市场与资产基础
-> F01 概率统计与数学基础
-> F05 固定收益与利率基础
-> Q01 风险指标实验 / Q03 定价实验
-> P03 fixed_income_task 候选
```

## 要解决的问题

- 债券为什么可以看成一组未来现金流？
- 现值、贴现率、到期收益率之间是什么关系？
- 为什么利率上升通常会让债券价格下降？
- duration 和 convexity 分别描述什么风险？
- FRED 这类公开利率数据如何进入实验？

## 学习目标

- [ ] 能写出最小债券现金流表。
- [ ] 能用折现公式计算债券现值。
- [ ] 能解释 coupon、maturity、yield、duration、convexity。
- [ ] 能用 Python 做一个利率变化对价格影响的小实验。
- [ ] 能说明利率曲线数据的来源和频率。
- [ ] 能把固定收益计算任务抽象成 P03 候选 task。
