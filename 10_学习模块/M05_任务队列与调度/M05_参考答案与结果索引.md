# M05 参考答案与结果索引

## 使用规则

本页只提供“完成学习者任务后去哪里对照”的映射，不复制完整答案、标准结果或可直接提交的
代码。学习顺序固定为：

```text
独立作答 -> 保存原始代码和结果 -> 运行验收 -> 再打开 reference -> 记录差异
```

P01 是 reference 真源。M05 教材负责解释概念，E05 负责学习者任务，本页只负责答案导航。
