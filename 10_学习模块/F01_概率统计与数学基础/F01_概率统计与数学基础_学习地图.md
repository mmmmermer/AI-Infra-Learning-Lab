# F01 概率统计与数学基础学习地图

## 怎么读这个模块

把 F01 当成金融工程实验需要的最低数学工具箱，不要当成纯数学大课。

阅读主线是：金融数据会波动，波动需要用随机变量、均值、方差、协方差和分布描述；组合风险需要用矩阵和优化描述；实验结论需要用统计检查和误差意识约束。

第一轮只学能支持 F02/F03 实验的内容，不追求证明完整性。

## 在总路线中的位置

F01 是金融工程支线的数学底座。

它连接：

```text
F00 金融市场与资产基础
-> F01 概率统计与数学基础
-> F02 金融时间序列
-> F03 投资组合与风险管理
-> M08/M11 指标解释和实验设计
```

它不替代 M01 Python 工程能力，也不替代 M11 科研方法。F01 只负责金融工程中最常用的概率、统计、线代和优化概念。

## 要解决的问题

- 为什么金融收益率要用均值、方差和分布描述？
- 协方差和相关性为什么能描述资产之间的共同波动？
- 为什么组合风险不是简单把单个资产风险相加？
- 置信区间、假设检验和统计误差对实验结论有什么影响？
- 组合优化为什么需要目标函数和约束？

## 学习目标

- [ ] 能解释随机变量、期望、方差、标准差、协方差。
- [ ] 能计算收益率序列的均值、波动率、相关矩阵。
- [ ] 能读懂协方差矩阵和相关性热力图。
- [ ] 能说明估计误差为什么会影响金融结论。
- [ ] 能理解最小优化问题：目标函数、变量、约束。
- [ ] 能为 F03 的组合风险实验做好数学准备。

## 第一轮学习产物

学完 F01 后，不要求写数学证明，只要求产出三件小东西：

1. 一页统计量说明：用自己的话解释 mean、variance、standard deviation、covariance、correlation。
2. 一张小样本计算表：用 5-10 个收益率样本手算或用 Python 计算均值、方差、波动率和相关性。
3. 一个组合风险前置说明：解释为什么 F03 需要协方差矩阵，而不是只看单个资产波动。

合格标准不是会推导公式，而是能把统计概念接到金融收益率和风险指标上。

## 推荐学习顺序

1. 先复习均值、方差、标准差、协方差和相关性。
2. 用收益率小样本手算一次均值和波动率。
3. 用 Python 计算协方差矩阵和相关矩阵。
4. 读 MIT 18.642 中和线代、概率、统计、数值方法有关的部分。
5. 查 SciPy optimize.minimize，理解优化器输入输出，不深入算法源码。
6. 回到 F03，看这些概念如何支撑组合风险。
