# F03 投资组合与风险管理学习地图

## 怎么读这个模块

把 F03 当成金融工程支线的第一个“风险解释与实验”模块，不要当成投资建议模块。

阅读主线是：从单资产收益和风险开始，理解多资产组合为什么受权重和相关性影响，再学习 Sharpe ratio、最大回撤、VaR 和有效前沿，最后把这些指标设计成未来 P03 的 `risk_metric_task` 或 `portfolio_task`。

第一轮只做可解释的风险指标和组合实验，不做实盘投资、不承诺收益。

正式学习时先读 [[10_学习模块/F03_投资组合与风险管理/F03_投资组合与风险管理_适配教材|F03 投资组合与风险管理适配教材]]，再做 GF03 实验。

## 要解决的问题

- 单个资产的收益和风险如何计算？
- 多资产组合收益为什么和权重有关？
- 多资产组合风险为什么还受相关性影响？
- Sharpe ratio、最大回撤、VaR 分别回答什么问题？
- 有效前沿是什么，为什么它不是投资建议？
- 风险计算任务如何设计成 P03 workload？

## 学习目标

- [ ] 能计算单资产收益率、波动率、最大回撤。
- [ ] 能计算多资产组合收益和组合波动。
- [ ] 能解释相关性对组合风险的影响。
- [ ] 能计算 Sharpe ratio、VaR、最大回撤等基础指标。
- [ ] 能画出两资产或多资产组合的风险收益散点。
- [ ] 能设计 `risk_metric_task` 和 `portfolio_task` 的输入输出。

## 第一轮学习产物

学完 F03 后，至少要能交出一个小型风险指标报告草稿：

1. 一张单资产风险表：return、volatility、max drawdown、VaR。
2. 一张两资产组合表：不同权重下的组合收益、组合波动和相关性说明。
3. 一张风险收益图：展示权重变化或多资产组合的风险收益散点。
4. 一个 P03 任务草图：定义 `risk_metric_task` 或 `portfolio_task` 的输入、输出、metrics、error_type。

合格标准不是找到“好组合”，而是能解释指标口径、数据限制和为什么这些结果不构成投资建议。

## 推荐学习顺序

1. 先完成 F02 的收益率和 rolling volatility 计算。
2. 复习 F01 的协方差、相关性和矩阵。
3. 阅读 MIT 15.401 中 Risk and Return、Portfolio Theory、CAPM 相关内容。
4. 用两个资产手算或脚本计算组合收益和组合风险。
5. 扩展到多个资产，画风险收益散点和简单有效前沿。
6. 设计 `risk_metric_task` 和 `portfolio_task` 的最小字段。

## 检查标准

- [ ] 能写出组合收益的计算方式。
- [ ] 能解释为什么相关性会影响组合风险。
- [ ] 能计算 Sharpe ratio、最大回撤、VaR。
- [ ] 能画出两资产组合风险收益曲线。
- [ ] 能解释 VaR 的局限。
- [ ] 能说明有效前沿不是收益承诺。
- [ ] 能设计 P03 风险指标任务的输入、输出和 metrics。
