# 库结构说明与检查清单

## 结构原则

这个 vault 不是纯资料库，而是围绕路线持续生产能力的系统。

核心链路：

```text
路线总控
-> 学习模块
-> 资料库
-> 知识卡片
-> 实验练习
-> 项目产出
-> 科研训练
-> 导师联系
-> 就业简历
-> 计划复盘
```
