这张表不是新教材，也不是项目成果，也不是要求你现在立刻跑实验。

尚未执行；只有你亲手运行代码、保存输出、填写记录表、写失败复盘并能解释指标口径后，才可以

## 当前使用规则

如果你还没有正式开始学 F 系列，本表只用于确认未来顺序，不用于催你现在做实验。

| 状态 | 现在做什么 | 不做什么 |
|---|---|---|
| 学习前准备 | 检查教材、资料、实验入口是否清楚 | 不跑 GF 实验，不创建 Q 项目 |
| 教材学习中 | 按 F00/F01/F02/F03 顺序读教材，整理概念和字段口径 | 不跳到后面实验，不写项目成果 |


## 执行前置

开始 F 系列实验前，不是只看会不会安装库，还要先完成对应教材阅读。最低前置是：

- 能创建并运行一个 Python 脚本。
- 能安装和导入 `pandas`。
- 能保存一份 `.py` 或 notebook 输出。
- 能把实验过程写成记录，而不是只看结果。
- 能明确所有金融实验都不构成投资建议。
- 已阅读对应模块的适配教材章节，能解释实验要验证什么。

建议先完成 M00/M01 的基础练习，再进入 F 系列实验。

## 实验环境准备

GF 实验按依赖库分三档。开始前先看清楚自己要跑的实验属于哪一档，避免照着代码跑却卡在 `ModuleNotFoundError`。

| 依赖档 | 需要安装 | 覆盖的实验 | 说明 |
|---|---|---|---|
| 标准库即可 | 无 | GF04-02、GF04-03 | 只用 `math`、`random`，任何 Python 都能跑 |
| 数据计算 | `pandas`、`numpy` | 第一阶段全部必跑实验（GF02/GF03 系列）、GF01-01、GF06、GF07-01、GF04-01 | 第一阶段地基，必装 |
| 画图（可选） | `matplotlib`、`seaborn` | GF01-02、GF03-01、GF03-02、GF04-01 的画图步骤 | 图是可选的；没有时只记录数据表即可，不影响实验结论 |
| 机器学习 | `scikit-learn` | GF07-02、GF07-03 | 第三轮才用到；`sklearn` 还会自动带上 `scipy`，无法用表格降级，必须装 |

第一阶段最小安装（只跑 GF02/GF03 必跑实验，足够开始）：

```bash
pip install pandas numpy
```

进入第二、三轮（回测画图、金融 ML）前再补：

```bash
pip install matplotlib seaborn scikit-learn
```

注意：

- 第一阶段必跑的四个实验只依赖 `pandas` / `numpy`，装好这两个就能开始，不用等装全。
- 画图类实验（GF01-02、GF03-01/02、GF04-01）如果没装 `matplotlib` / `seaborn`，跳过画图、只填数据表即可，验收标准照样能满足。
- 只有 GF07-02 / GF07-03 真正离不开 `scikit-learn`：它是实验核心步骤，不能用记录表替代。跑这两个前务必先 `pip install scikit-learn`。
- 不确定某个库装没装，先在 Python 里 `import pandas`（或对应库名）试一下，能导入再跑实验。

## 第一阶段教材阅读顺序

第一阶段先学教材，再触发实验。不要把 GF 实验当成学习起点。

推荐阅读顺序：

```text
F00 金融市场与资产基础
-> F01 概率统计与数学基础
-> F02 Python 金融数据与时间序列
-> F03 投资组合与风险管理
```

每个模块的阅读目标：

| 模块 | 阅读目标 | 读完后应该能说清 |
|---|---|---|
| F00 | 建立资产类别、数据来源和字段口径 | 价格、收益率、利率、财报、宏观序列分别是什么 |
| F01 | 建立收益率样本、分布、分位数、协方差直觉 | 均值/波动/VaR/相关性为什么依赖样本和口径 |
| F02 | 学会金融时间序列清洗和收益率计算 | 日期索引、缺失值、复权字段、rolling 指标如何记录 |
| F03 | 学会风险指标和组合风险解释 | VaR、max drawdown、权重、协方差和组合风险的边界 |

## 实验触发规则

只有读到对应章节并能解释“为什么做这个实验”时，才进入 GF 实验。

| 触发点 | 可以做的实验 | 触发条件 |
|---|---|---|
| 读完 F00 + F02 前 3 章 | GF02-01 金融时间序列清洗 | 能解释数据来源、字段口径、日期索引、缺失值 |
| 读完 F02 收益率/rolling/drawdown 章节 | GF02-02 rolling volatility 与最大回撤 | 能解释 return、rolling window、drawdown |

没有达到触发条件时，只读教材和补概念，不跑实验。
