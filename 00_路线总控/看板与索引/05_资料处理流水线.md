## 标准流程

```text
外部资料
-> 20_资料库：保存来源和摘要
-> 10_学习模块：放入学习路径
-> 30_知识卡片：提炼概念或机制
-> 40_实验练习：设计动手任务
-> 50_项目产出：落到项目功能
-> 60_科研训练：形成问题、实验或论文笔记
-> 80_就业市场与简历：转成岗位语言
```

## 资料进入规则

每条资料必须回答：

- 它属于哪个学习模块？
- 它解决哪个问题？
- 它是必学、选学还是参考？
- 它能转化成什么实验或项目功能？
- 是否能帮助科研或就业表达？

## 不直接塞进资料库的内容

- 低质量搬运文章
- 没有上下文的视频链接
- 只讲概念不讲实现的碎片
- 和当前路线无关的热门资料

## 资料状态

建议在资料卡里标记：

- `候选`
- `必读`
- `选读`
- `查阅`
- `已转化`
- `暂缓`
