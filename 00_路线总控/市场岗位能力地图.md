# 市场岗位能力地图

## 目标岗位族

- AI 应用服务端工程师
- RAG/Agent 工程师
- LLMOps / MLOps 工程师
- AI Platform 工程师
- Inference Infra 工程师
- 云原生平台开发
- 金融科技 AI 平台 / 投研工具工程师

## 岗位共性需求

### AI 应用落地

- RAG
- Agent workflow
- prompt / tool calling
- 向量数据库
- 文档解析
- 权限过滤

### 后端工程

- API 设计
- 数据库
- Redis / 缓存
- 异步任务
- 日志与异常处理
- 测试
- Docker 部署

### 平台与云原生

- Docker / Kubernetes
- 服务发现
- 扩缩容
- 资源隔离
- 队列和配额
- 多租户

### 可观测性

- metrics
- logs
- tracing
- Prometheus / Grafana
- 压测
- P95 / P99
- failure triage

### 推理服务

- vLLM
- Triton
- Ray Serve
- batching
- KV cache
- tokens/s
- TTFT / TPOT

### 安全与治理

- 用户权限
- 数据隔离
- prompt injection
- 敏感信息保护
- 文件上传安全
- 审计日志
