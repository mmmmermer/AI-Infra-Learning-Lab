# 资料筛选与学习规则

## 目标

资料库不是收藏夹。每条资料都必须能服务于学习模块、实验、项目、科研问题或就业表达。

## 收录优先级

1. 官方文档
2. 顶级高校公开课
3. CNCF / 云原生项目官方资料
4. 经典论文或项目设计文档
5. 少量高质量工程实践教程

## 不收录

- 来源不明的二手搬运
- 只讲概念但没有工程落点的文章
- 和当前路线无关的热门资料
- 无法转化成实验、项目或知识卡片的资料

## 每条资料必须标注

- 资料名称
- 来源链接
- 类型
- 适合阶段
- 对应模块
- 对应实验或项目
- 当前状态

## 状态定义

- 候选：真实存在，适合路线，但尚未开始处理
- 必读：当前阶段必须用
- 选读：遇到问题或进阶时再读
- 查阅：作为工具文档使用
- 已转化：已经转成知识卡片、实验记录或项目功能
- 暂缓：资料有价值，但当前阶段不适合投入时间

## 学习规则

- 每周最多激活 3-5 条资料。
- 官方文档按任务查，不追求从头看到尾。
- 高校课程只选和当前实验相关的章节。
- 论文只在工程问题已经出现后再读。
- 每条必读资料至少转化成一个输出：知识卡片、实验记录、项目功能或 README 片段。
